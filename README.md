# kotoba-lang/kotodama-holochain

Holochain agent actor runtime payload examples extracted from the legacy
`kotoba-kotodama` tree.

This is separate from the Kotoba language/runtime repository and from the
Kotodama host SDK.

The former Rust zome scaffold, Cargo workspace, Docker build scaffold, and smoke
client have been removed. If Holochain support is revived, implement it in an
adapter-owned repository that consumes the JSON/Kotoba contracts kept here.
